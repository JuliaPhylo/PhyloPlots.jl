## overview

PhyloPlots has utilities to plot tree and network objects
from [PhyloNetworks](https://github.com/crsl4/PhyloNetworks.jl)
through [R](https://www.r-project.org)
via [RCall](https://github.com/JuliaInterop/RCall.jl).
